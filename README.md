# student-grades-calculator
A simple Python program that collects students’ grades, calculates their averages, and identifies the best and worst students.
A simple Python script to manage students’ grades.  
It lets you:
- Enter grades for multiple students.
- Automatically calculate each student’s average (Moyenne).
- Display all students’ grades in a clear format.
- Identify the 🏆 **best student** and the 📉 **worst student** based on their averages.

---

## 🚀 Features
- Input validation to ensure grades are between 0 and 20.
- Automatic calculation of each student’s average (rounded to 2 decimals).
- Finds the highest and lowest averages.
- Easy to run in any Python environment.
